## @fun-gauge/react:

 - 120 fps
 - lightweight 7.5kB min gizp
 - customization
 - responsive design
 - typescript


## How to use?

<Steps>

1. Install '@fun-gauge/react':


    <Tabs>
        <TabItem label="npm">
            ```sh
            npm install @fun-gauge/react
            ```
        </TabItem>
        <TabItem label="pnpm">
            ```sh
            pnpm add @fun-gauge/react
            ```
        </TabItem>
        <TabItem label="yarn">
            ```sh
            yarn add @fun-gauge/react
            ```
        </TabItem>
    </Tabs>


2. Use the component:

    ```javascript
    import Gauge from '@fun-gauge/react'

    function MyGauge(props) {
        const { title = 'Hello Gauge', value = 60 } = props

        return (
            <div>
                <h4>{title}</h4>
                <Gauge value={value} />
            </div>
        )
    }
    ```

</Steps>
